# PRD — thierry-laptop

## O que é este projeto

Documento de análise e guia de compra de portátil criado para o Thierry (irmão do Jeremy), publicado como repositório público para que ele possa consultar autonomamente.

## Porquê existe

O Thierry quer comprar um portátil com critérios específicos de reparabilidade e upgradeabilidade que vão além do que uma pesquisa normal no Google responde facilmente. Este guia consolida pesquisa actualizada (2025/2026) num formato claro e acessível.

## O que é

Um único ficheiro HTML (`index.html`) com:
- Análise de critérios de compra
- Explicação da realidade sobre CPUs soldadas vs. socketed
- Nota educativa sobre WiFi 2.4 GHz vs. 5 GHz
- Requisitos mínimos para IBM SPSS e Excel
- Análise detalhada de 5 modelos com pros/cons
- Tabela comparativa
- Recomendação final fundamentada
- Glossário de termos técnicos

## O que NÃO é

- Não é uma loja nem tem links de afiliados
- Não é um documento vivo de preços (preços mudam com o tempo)
- Não cobre portáteis gaming nem ultrabooks finos e leves

## Utilização prevista

O Thierry abre `index.html` no browser (ou via GitHub Pages no futuro) e navega pelas secções que lhe interessam. Não requer internet para funcionar — é um ficheiro HTML standalone.
