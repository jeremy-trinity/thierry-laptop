# thierry-laptop

Guia de compra de portátil para o Thierry — análise de modelos eficientes, duráveis, reparáveis e com capacidade de upgrade (2025/2026).

## Acesso

Abre o ficheiro `index.html` diretamente no browser para ver o guia completo.

## Critérios analisados

- CPU não soldada (socketed) — realidade do mercado explicada
- RAM em 2 slots SO-DIMM (não soldada)
- 2 slots M.2 para armazenamento
- Bateria substituível pelo utilizador
- Placa WiFi M.2 substituível/upgradeable (2.4 GHz e 5 GHz)
- Carregamento via USB-C com suporte fast charging
- Ecrã OLED de qualidade
- Capacidade para correr IBM SPSS e Microsoft Excel

## Modelos analisados

1. Lenovo ThinkPad T16 Gen 5 — Recomendação principal
2. Framework Laptop 16 (2025) — Máxima modularidade
3. HP EliteBook 860 G11 — Alternativa profissional premium
4. Dell XPS 15 (9530/9540) — OLED excelente mas WiFi soldado
5. Eurocom Raptor X16 — CPU socketed mas impraticável pelo preço/peso
